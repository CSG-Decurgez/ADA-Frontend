Introducción a los selectores avanzados

Los selectores avanzados permiten aplicar estilos más específicos y potentes en CSS. A continuación veremos ejemplos prácticos de cada tipo.

Selectores de atributos


/* [atributo] — Selecciona elementos que tienen un atributo */
input[type] {
  border: 2px solid steelblue;
}

/* [atributo="valor"] — Valor exacto */
input[type="email"] {
  background-color: #eef;
}

/* [atributo~="valor"] — Contiene una palabra */
[class~="destacado"] {
  color: crimson;
}

/* [atributo^="valor"] — Comienza con */
a[href^="https://"] {
  font-weight: bold;
}

/* [atributo$="valor"] — Termina con */
img[src$=".png"] {
  border: 3px solid green;
}

/* [atributo*="valor"] — Contiene */
a[href*="example"] {
  color: orange;
}

Selectores avanzados para jerarquías


/* Hermano adyacente (A + B) */
h1 + p {
  color: teal;
}

/* Hermanos generales (A ~ B) */
h1 ~ p {
  font-style: italic;
}

/* Descendiente (A B) */
div p {
  margin-left: 20px;
}

/* Hijo directo (A > B) */
ul > li {
  background-color: #f5f5f5;
}

Selector universal *


/* Aplica a todos los elementos */
* {
  box-sizing: border-box;
  margin: 0;
  padding: 0;
}

Pseudo‑clases


/* ::before y ::after */
h1::before {
  content: "★ ";
}
h1::after {
  content: " ★";
}

/* :not() */
button:not(.primario) {
  background-color: lightgray;
}

/* :nth-child() */
li:nth-child(2) {
  color: purple;
}

/* :nth-of-type() */
p:nth-of-type(2) {
  font-weight: bold;
}

/* :hover */
a:hover {
  color: tomato;
}

/* :active */
button:active {
  transform: scale(0.95);
}

/* :focus */
input:focus {
  outline: 2px solid royalblue;
}

/* :target */
#seccion1:target {
  background-color: yellow;
}

/* :checked */
input[type="checkbox"]:checked {
  accent-color: green;
}

/* :valid / :invalid */
input:valid {
  border-color: green;
}
input:invalid {
  border-color: red;
}

/* :placeholder-shown */
input:placeholder-shown {
  color: gray;
}
