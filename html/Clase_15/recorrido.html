Introducción

Este recorrido resume todo lo que aprendimos desde la primera clase hasta la quince. Es una mirada general para ver el progreso, conectar conceptos y entender cómo cada tema construyó el siguiente.

Clase 1 — Introducción al Desarrollo Web

Qué es la web, cómo funciona un navegador y estructura mínima de un archivo HTML.

Clase 2 — HTML básico

Etiquetas esenciales: títulos, párrafos, listas, enlaces. Primeros atributos.

Clase 3 — Imágenes y vínculos

Uso de <img>, rutas relativas/absolutas y enlaces internos/externos.

Clase 4 — Estructura semántica

Uso de <header>, <main>, <section>, <article>, <footer>.

Clase 5 — Introducción a CSS

Primeros estilos, selectores básicos, colores, tipografías.

Clase 6 — Box Model

Margin, padding, border, content. Todo en la web es una caja.

Clase 7 — Display y flujo

Block, inline, inline-block y cómo fluye el documento.

Clase 8 — Selectores y especificidad

Cómo CSS decide qué regla gana. Buenas prácticas para evitar conflictos.

Clase 9 — Clases y reutilización

Uso de clases, modularidad, evitar IDs, evitar selectores complejos.

Clase 10 — Tipografías y unidades

px, rem, em, %, vh, vw. Importación de fuentes.

Clase 11 — Colores, fondos y bordes

Paletas, degradados, imágenes de fondo, contraste.

Clase 12 — Posicionamiento

relative, absolute, fixed, sticky, z-index.

Clase 13 — Posicionamiento aplicado

Badges, overlays, tarjetas. Ejercicio guiado.

Clase 14 — Transiciones y transformaciones

transition, transform, cubic-bezier, microinteracciones.

Clase 15 — Buenas prácticas con clases

Nombres descriptivos, kebab-case, consistencia, estados, modularidad.

Cierre del recorrido

Empezamos con un archivo vacío y hoy ya podemos construir, estructurar, estilizar, animar y organizar un sitio completo con buenas prácticas.

Lo que viene ahora es mostrarlo al mundo.
