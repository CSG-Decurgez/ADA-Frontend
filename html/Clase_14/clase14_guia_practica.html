Introducción

Las transiciones y transformaciones en CSS permiten animar y cambiar el estado de los elementos de manera suave y atractiva. Aquí veremos ejemplos prácticos de transiciones, timing functions, transformaciones 2D y 3D.

Transición básica (background + scale)

Hover me
<div class="box box-transition">Hover me</div>

/* CSS */
.box-transition {
  transition-property: background-color, transform;
  transition-duration: 300ms;
  transition-timing-function: ease;
}
.box-transition:hover {
  background-color: #ff6b6b;
  transform: scale(1.06);
}

Propiedades de transición (property, duration, timing, delay)

/* shorthand */
.btn-transition {
  transition: background-color 400ms ease-in-out 100ms;
}
.btn-transition:hover { background-color: #2ecc71; }
