¿Qué es el posicionamiento en CSS?

El posicionamiento en CSS es el mecanismo que nos permite controlar dónde aparece un elemento en la página y cómo se relaciona con los demás elementos. Por defecto los elementos siguen un flujo normal.

Bloque A
Bloque B
<div class="demo-box">Bloque A</div>
<div class="demo-box">Bloque B</div>

¿Para qué sirve el posicionamiento?

Mover elementos, superponer, fijar en pantalla y controlar orden visual con z-index.

Base
Top (z-index mayor)
.demo-card.top { position: relative; z-index: 10; }

Posicionamiento normal static

Comportamiento por defecto: elementos en flujo normal.

Item 1
Item 2
.demo-static-item { display: block; }

Posicionamiento relativo relative

Ocupa su lugar en el flujo pero puede moverse visualmente.

Caja relativa
.demo-rel { position: relative; top: 10px; left: 20px; }

Posicionamiento absoluto absolute

Se posiciona respecto del ancestro posicionado y sale del flujo.

Caja absoluta
Contenedor
.demo-abs-wrapper { position: relative; }
.demo-abs { position: absolute; top: 10px; right: 10px; }

Posicionamiento fixed

Se posiciona respecto del viewport y permanece al hacer scroll.

(Busca el botón amarillo fijo en pantalla)

.demo-fixed { position: fixed; bottom: 20px; right: 20px; }

Sistema de coordenadas

Propiedades top, right, bottom, left y z-index.

Coordenadas
.demo-coord { position: absolute; top: 30px; left: 40px; }

Elementos flotantes float y clear

Float permite columnas; clear recupera el flujo.

Columna 1
Columna 2
.demo-float-col { float: left; width: 50%; }

Concepto de grilla

Dividir en columnas (ej. 12) para componer layouts.

A
B
C
.col { float: left; width: 33.33%; }
