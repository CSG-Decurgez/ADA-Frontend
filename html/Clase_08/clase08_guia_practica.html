1. Meta viewport

Para que el diseño responda correctamente en dispositivos móviles, agregamos esta etiqueta en el <head>:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Esto evita zooms automáticos y asegura que el ancho del sitio coincida con el ancho del dispositivo.

2. Unidades relativas: %, vh, vw

% depende del contenedor padre. vh y vw dependen del tamaño de la pantalla.

50% del ancho
30vh de alto
40vw de ancho
/* clase08.css */
.porcentaje { width: 50%; background: #CD0055; color: white; }
.vh { height: 30vh; background: #555; color: white; }
.vw { width: 40vw; background: #777; color: white; }

3. Unidades relativas: em y rem

em depende del tamaño de fuente del contenedor. rem depende del tamaño raíz.

Este texto usa 1.5rem.

Este contenedor tiene font-size: 1.2em.

Este texto hereda ese tamaño.

/* clase08.css */
.texto-rem { font-size: 1.5rem; }
.contenedor-em { font-size: 1.2em; background: #eee; padding: 1rem; }

4. Cálculos con CSS

Podemos combinar unidades con calc():

.caja-calc {
  width: calc(100% - 2rem);
  padding: 1rem;
  background: #CD0055;
  color: white;
}
Este ancho usa calc().

5. Imágenes responsivas

1) Imagen que se adapta al contenedor

img { max-width: 100%; height: auto; }

2) Usando srcset (demostración)

<img 
  src="image1-400.jpg"
  srcset="image1-800.jpg 800w, image1-1200.jpg 1200w"
  alt="Ejemplo">

3) Usando picture (demostración)

<picture>
  <source media="(min-width: 800px)" srcset="image1-large.jpg">
  <img src="image1-small.jpg" alt="Ejemplo">
</picture>

6. Media Queries

Trabajamos con enfoque Mobile First:

/* Estilos base (móvil) */
body { font-size: 1rem; }

/* Tablet */
@media (min-width: 768px) {
  body { font-size: 1.2rem; }
}

/* Desktop */
@media (min-width: 1024px) {
  body { font-size: 1.4rem; }
}

Probá cambiar el tamaño de la ventana para ver el efecto.

7. Grid responsivo

Una galería que se adapta automáticamente:

.gallery {
  display: grid;
  gap: 1rem;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
}

10. Width vs Max Width

A partir de la consulta en clase de cual era la diferencia entre usar por ej. width: 100% o max-width: 100% les dejo un ejemplo con una imagen.

¿Qué muestra este ejemplo?

Escenario 1 — Contenedor grande
- width: 100% → **estira** la imagen
- max-width: 100% → **respeta el tamaño real**

Escenario 2 — Contenedor chico
- Ambas se achican → **la imagen se comprime proporcionalmente**

La diferencia sólo aparece cuando el contenedor es más grande que la imagen.
Si el contenedor es más chico, ambas reglas se comportan igual.
